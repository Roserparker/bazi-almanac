# 天人之学 ·「简单信息分析」设计 spec

> 经 superpowers 头脑风暴产出、用户已确认（"可以 去做吧"）。
> 立心：一个生辰 / 一个日子 = **同一片天**上的一个坐标。各文明用不同格子切这片天、读人事——中（节气/干支/生肖）、西（12 宫×30°）、埃及（36 旬星×10 天）。**八字是深核心，西/埃作"对照之镜"，统一在"同一片天 · 道通为一"。**

## 中枢：万年历「简单信息分析」面板（选定日，随导航实时更新）
分三块，由近及远：

### ① 时间层叠加（录生辰后个性化）—— 先做
随选定日显示四层（均相对日主）：
- **大运**：覆盖该年的那步（干支 + 起讫年龄 + 十神 + 喜/忌）。
- **流年**：干支 + 十神 + 喜/忌。
- **流月**：干支 + 十神 + 喜/忌。
- **流日**：干支 + 十神 + 顺/逆。
- **叠加基调**：四层之气落喜用多还是忌神多 → 整体偏顺 / 偏逆 / 参半。
- 降级：未录生辰时只显客观干支 + 节气/生肖/宜忌，不显十神/喜忌/大运。

### ② 历法本源（深化节气 + 生肖）
- **节气** + 七十二候物候；点破"节气 = 月令换柱的开关"（直接驱动旺衰/用神）。
- **生肖** = 年支动物；连到地支六合 / 六冲（生肖配对 / 相冲的真相）。

### ③ 对照之镜（中 / 西 / 埃 · 道通为一）
- **西方太阳星座**（公历日期算）+ 它怎么切天（12 宫×30°）。
- **古埃及 36 旬星 decan**（公历算；用 12 宫×3 decan 的旬星系统，每 decan ≈10 天）+ 星/神 + "10 天一旬 ≈ 中国的候"。
- 一句"同一片天、三种切法"的连接语。

## 约束
- 全部从**公历确定性算出**；本地、大陆友好、零新依赖。
- 西/埃只做**对照 + 教学**，不做深度预测、不稀释八字。
- 八字仍是深核心。

## 待建数据

- **古埃及 36 旬星表**（12 宫×3 decan：名/星/神/度数）——需整理，有据可考（Dendera / decanate 列表）。放 ③ 实现时做。

## 实现顺序
① 时间层叠加（数据全现成）→ ② 历法本源 → ③ 对照之镜（埃及旬星表整理后）。
