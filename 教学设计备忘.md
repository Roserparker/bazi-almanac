# 教学设计备忘 · 化机历「教育板块」

> 经一轮头脑风暴产出的教学法设计（spec）。先据此评审/确认，再实现。
> 用户定调：**以通用知识为骨架（教科书），淡化"我的命盘"，只偶尔拿出来举例、抛砖引玉**——目的是学新知识，不是算自己的命。掌握好比例与分寸。

## 0. 两条统领主张
1. **日主主线**：一切关系都回答一句"对日主是帮还是耗？怎么帮 / 怎么耗"。合冲刑害克全挂到这条主线上，就不散、不吓人。
2. **去恐吓化 + 渐进**：福祸相依，没有纯凶；每个概念先一句话，可展开看深层。小白不被淹没，进阶能挖深。

## 1. 悬停 / 讲解的「分层模型」（核心）
对 合 / 冲 / 刑 / 害 / 克，统一分 **4 层**，默认显示前 2 层，"展开"看 3–4：

| 层 | 内容 | 例（乙庚合 / 子午冲） | 对谁最重要 |
|---|---|---|---|
| **L1 是什么** | 一句话本质 | 六合＝两支结盟相吸；相冲＝两气正面相撞 | 小白必看 |
| **L2 怎么合 / 怎么克（五行机理）** | 具体到这两个字 | 乙庚＝阴木阳金相吸、合化金；子午＝水火相格（水克火） | 小白必看 |
| **L3 对日主 / 十神的意义** | 帮还是耗、点亮哪个十神 | 挂回主线 | 进阶（用起来） |
| **L4 实战与暗机** | 明克暗克、冲中带合、合化解克、该动则吉/该静被冲则波澜 | 暗线 | 高阶 |

→ 初学者最重要的是 **L1、L2**（认得＋懂机理）；L3 是"用"，L4 是暗机。
→ 现有 `interpret.explainRelation` 已覆盖 L1–L3 雏形；要重构成"分层、可展开"。

## 2. 明克 vs 暗克（用户特别问）
- **明克（显）**：天干照面直克（丙火克庚金）；地支正冲（子午、卯酉）。一眼可见。
- **暗克（隐）**：
  1. **藏干暗克**：地支藏干之间的克（午藏丁火 暗克 申藏庚金；表面午申无事，藏干里火金交战）。
  2. **入墓 / 冲库**：被克入墓、冲库引动。
  3. **隔位 / 遥克**：隔柱的弱化之克。
  4. **贪生忘克**：有通关之神，克被化（金克木，但中间有水 → 金生水、水生木，克解）。
  5. **合化解克 / 合绊**：相合把该克的力"合住"。
- **教法次第**：先教明克（看得见）→ 再教"地支拆开看藏干"（暗克之源）→ 最后教"通关/合化"如何化解。这正是从"八个字"到"看懂暗线"的关键跃迁。

## 3. 「找到一个八字的运行逻辑」——可教的 5 步（用户特别问）
把"一堆干支"变成"看懂主线"的方法，做成**交互式分步 walkthrough**（默认用通用例盘；可选"换成我的盘"作一次性举例）：
1. **定日主**：日柱天干＝你自己，记其五行阴阳。
2. **看月令断旺衰**：日主生于何月（月支当令否）→ 身强 / 身弱（粗略）。
3. **分十神**：其余干支相对日主是什么十神 → 各管什么领域。
4. **找用忌**：身强喜克泄耗、身弱喜生扶 → 哪种五行是"药"（用神）、哪种是"病"（忌神）。
5. **看引动**：大运 / 流年 / 流日 来字，生扶用神＝顺、助忌＝逆；冲合刑害引动哪一柱。
→ 这 5 步＝"运行逻辑"。做成可点的流程卡。（依赖 M6 身强弱/用神，先出框架、判定标"参考"。）

## 4. 工作坊分区（scaffolding 次第）
每个工作坊 = 简短课文 + 图 + 对应 Anki 卡组；概念词皆可悬停/点开看分层。顶部加导航在板块间跳转。

- **W0 入门心法**：阴阳、五行是什么、日主主线、福祸相依立场。（最短，先立心态）
- **W1 五行**：木火土金水体性（依五行大义）、相生相克、四时旺衰、五行流转环。
- **W2 干支系统**：十干 / 十二支 / 藏干 / 纳音 / 生肖；明的合冲刑害。
- **W3 八卦易理**：太极→两仪→四象→八卦、后天八卦方位图、洛书九宫、八卦配五行。
- **W4 十神**：十神是什么、怎么从日主推、每神两面性。
- **W5 八字计算方式**：四柱怎么排（年立春 / 月节气 / 日六十甲子 / 时五鼠遁）、大运起法、流年月日。
- **W6 读盘逻辑（进阶）**：第 3 节的 5 步、旺衰、用忌、暗克、引动。

现有 `#learn`（八字入门）并入 W0/W4；`#yixue`（八卦）并入 W3。命盘/万年历＝"私人用"区，与"学习"区分开但互通。
