## 二、紫微斗数板块（#ziwei，录入生辰后出现）

- **安星法**：《紫微斗数全书》通行诀（三合派）。闰月归本月；时辰取生时支（未录时间默认 12:00=午时，界面注明）。
- 范围：命宫/身宫（寅起正月，逆数时/顺数时）→ 五虎遁宫干 → 命宫纳音定**五行局** → 安**紫微**（局数商余、奇退偶进）→ 紫微星系 6 星（机阳武同廉）→ 天府 =（16−紫微）mod 12 → 天府星系 8 星 → 辅弼（月系）/昌曲（时系）/禄存羊陀·魁钺（年干）/火铃（年支三合+时）/空劫（时）/天马红鸾天喜（年支）→ **生年四化**（庚干从「阳武阴同」，界面注明流派）→ 命主/身主。
- UI：4×4 宫格（12 宫环列，中央 2×2 放命造摘要：五行局/命主身主/生年四化）。主星带两面性贴士（去恐吓化：贪狼=欲望也是生命力，七杀=将星…），今日**流日四化**落宫高亮。
- **流曜**：流年支=流年命宫；流年/流月/流日**天干四化**各自落宫 → 供指数 v2 与紫微卡「今日四化」行。

## 三、化机指数 v2（daily.js 升级，两处 UI 同步）

v1 = 流日契合 50% + 层运共振 28% + 合冲动静 22% (+调候 6)。v2 细化为五因子：

| 因子 | 权重 | 算法 |
|---|---|---|
| 流日契合 | 32% | 沿用 dayHit（干 0.6/支 0.4，调候字命中） |
| **五行能量** | 20% | **新**：energyProfile —— 流日 0.5/流月 0.3/流年 0.2，干 0.6 + 藏干 0.4（本中余 1/0.5/0.3 归一），再乘**月令旺相休囚死系数**（1.25/1.1/0.9/0.75/0.6）→ 归一成五行能量百分比谱；投影到（喜用−忌神）轴得 [-1,1] |
| 层运共振 | 22% | 沿用（大运/流年/流月），UI 拆出每层小值 |
| 合冲动静 | 16% | 沿用 relMotion |
| **紫微流曜** | 10% | **新**：流日干四化——化禄/权/科落命之三方四正（命财官迁）各 +0.5/+0.3/+0.3，化忌落三方四正 −0.6、落命宫再 −0.3，clamp [-1,1]；无紫微盘（未录生辰的 BTC 盘也有）则该项 0 且权重并回流日契合 |

调候日 +6 与分档（昂扬/顺畅/平稳/收敛/蛰养）不变。UI：仪表盘/选定日面板显示 5 条因子横杆 + 层运细分 + **五行能量谱**（五色小柱，喜用标 △ 忌标 ▽）。HINT_BASE.化机指数 文案与断言同步更新。

## 四、BTC 观象台（#btc，无需录生辰即可看）

- **BTC 命**：创世块 2009-01-03 18:15:05 UTC = **北京时间 2009-01-04 02:15**（师承匿名博士 Chris_Defi 的通行锚点，界面注明换算与主观性）→ **戊子 甲子 己酉 乙丑**，日元**己土**（田园之土），子月**身弱财旺**，**喜火土 / 忌金水木**，调候丙甲戊。已与本站引擎复算一致。
- **五行属性**：主判「己土」；旁注民间「数字黄金=金」说并说明本站取日元说。
- **时代层**：不用个人大运（性别顺逆对资产无意义），改用**三元九运**：八运艮土 2004–2023 → **九运离火 2024–2043**（火=印星，生弱土——「火运生身」为长期底色）。指数层运项以「丙午」代表九运入模，标注为象征映射。
- **今日走向**：dayIndex v2（BTC 盘 + BTC 紫微盘流曜）→ BTC 化机指数 + 流日与 BTC 盘合冲刑害列表（不带宫位尾注——宫位义为人设）。
- **本周走向**：今起 7 日指数曲线（SVG sparkline，节点按档色），配一句周评（高低日/节奏：先扬后抑…）。

## 五、每日一卦 · 六爻（js/liuyao.js，挂在观象台内）

- **成卦**：以日期字符串 `YYYY-MM-DD#BTC` 哈希做种（mulberry32），三枚铜钱掷六次（6 老阴/7 少阳/8 少阴/9 老阳），自下而上成**本卦**，动爻翻转得**变卦**。同日全网稳定、次日更新。
- **装卦**：京房纳甲（八宫纳干支表）→ 六亲（以卦宫五行论生克：生我父母/我生子孙/克我官鬼/我克妻财/同我兄弟）→ **世应**（八宫卦变算法推宫与世位，非查表）→ 六神（按流日天干起青龙…顺布）。
- **断 BTC（问财，用神=妻财爻）**，简化规则记分并列出依据：月建/日建对财爻生扶或克（±2/±1.5）、财爻发动（+1，化回头克 −1.5）、子孙动（原神 +1.5）、兄弟动（劫财 −1.5）、世爻旺衰（承接力 ±0.5）→ 「偏扬 / 震荡 / 偏抑」+ 强度。
- 数据：64 卦名 + 卦辞（原文节选）+ 象曰一句 + 白话一句。UI：卦画（动爻标 ○×）、本卦→变卦、爻侧纳甲/六亲/世应/六神、断语 + 依据。

## 六、每日奇门 · 午时盘（js/qimen.js）

- **时家奇门 · 转盘 · 拆补法**：当日**午时**为盘（日中，标注取法）。节气（lunar getPrevJieQi 全 24）定阴阳遁与三局歌诀表（冬至惊蛰一七四…）；日柱符头（甲己）定上中下元（子午卯酉上/寅申巳亥中/辰戌丑未下）。
- 排盘：地盘三奇六仪（阳顺阴逆）→ 时柱旬首定**值符星/值使门** → 九星随时干转宫、八门按时支数宫（中五寄坤）、八神阳顺阴逆。
- **读 BTC（问财）**：看**生门**与**戊**（资本）——生门临戊/相生=资金活络，生门落宫旺相（得月令）加分，门迫/击刑减分 → 两三条判读 + 倾向。UI：3×3 九宫格（神/星/干/门），值符值使行，判读行。
- 六爻与奇门**并列呈现**（用户要求「都给出来看看效果」），各自独立结论，允许相左——注明「两把尺子，参差本是常态」。
