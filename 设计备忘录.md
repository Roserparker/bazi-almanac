# 化机历 · 设计备忘录

> Working title「化机历」（化 = 化学反应 / 化气，机 = 天机 / 玄机；可改）。
> 一句话：输入生辰 → 浏览器本地自动排八字 → 把「今天 / 本月 / 今年 + 当前大运」的干支能量与你的本命盘做一次"化学反应"，给出克制、可读、不贩卖焦虑的每日运势解读。

---

## 0. 价值主张 / 与市面黄历的区别
- 市面黄历：千人一面的"宜嫁娶 忌动土"，靠红金弹窗与吉凶恐吓。
- 化机历：基于**你个人八字**的个性化能量分析；调性素静、文人、反珠光宝气；像一本安静的杂志 / 手账。
- 隐私即卖点：所有排盘与解读在**浏览器本地**完成，生辰数据不出本机、不上传。

## 1. 命理理论基础（参考书）
- 《滴天髓》：以日主为核心，讲五行生克、旺衰、调候、通根透干、病药 —— 本项目"能量化学反应"的主轴。
- 《三命通会》：十神、神煞、格局、大运流年的体系化论述 —— 术语与意象来源。
- 立场：把它们作为**结构化的传统模型与文化意象**来用，给出的是"参考视角"而非宿命断言。文案克制、可读、不迷信。

## 2. 命理引擎（不重造天文轮子）
- 历法 / 节气 / 排盘交给成熟库 **lunar-javascript**（6tail）。它处理最"硬"的部分：公农历转换、24 节气定月、真太阳时、六十甲子日柱、五鼠遁时柱、大运起运方向与岁数、十神、藏干、纳音、长生十二宫。
- 我们自研**解读层**：在准确的干支结构之上做"化学反应"分析与中文文案。
- 引擎输出（本命盘）：
  - 四柱八字（年 / 月 / 日 / 时 干支）
  - 五行分布与旺衰（结合月令旺相休囚死）
  - 十神（比劫 / 食伤 / 财 / 官杀 / 印）
  - 地支藏干、纳音、日主长生状态
  - 日主强弱（身强 / 身弱）、用神·喜神·忌神（**初版规则估算，明确标注"参考"**，二期精修）
- 大运：起运方向与岁数、各步十年大运干支。
- 流年 / 流月 / 流日：当下干支链。

## 3. 「能量化学反应」模型（核心）
把"化学反应"这个比喻落到确定性规则上：

**反应物**：本命四柱（叠加当前大运）的五行 / 十神格局 ＋ 当下 流年 → 流月 → 流日 的干支。

**化学键（互动类型）**：
- 天干：五合（甲己合土…）、相生、相克。
- 地支：六合、三合局、三会、六冲、相刑、相害、相破。
- 十神触发：今日干支落在你命盘的哪个十神 = 今天被"点亮"的能量类型（事业 / 财 / 关系 / 表达 / 内省）。
- 用忌判断：今日五行**生扶用神**（顺）还是**助长忌神**（需留意）。
- 旺衰叠加：流月令的旺相休囚 ＋ 今日。

**反应产物（给用户看的）**：
- 一句话主题（今天哪类能量活跃）。
- 一段克制的展开：宜做什么 / 留意什么，具体而非吓人。
- 一个温和的"能量强度 / 顺逆"指示（柔和刻度，不是吉凶大字、不是分数恐吓）。
- 嵌入《滴天髓》《三命通会》的术语 / 意象（通根、调候、病药…）作风味，但**用白话解释，不掉书袋**。

## 3.5 教学与去恐吓化（核心价值）
- 每个术语都「点开能学」：中性本质 → 两面性（吉凶各在什么条件）→ 如何顺势用好 →《道德经》注脚（福祸相依）。
- 立场：不吓人、不宿命。劫财、七杀、伤官、偏印这些被市面妖魔化的「凶神」，重点平反——同一股能量，身强身弱、用法不同，福祸就不同。
- 内容存于 `js/knowledge.js`（纯数据），UI 点击术语弹浮层；首页底部有「八字怎么运作」入门。

## 4. 信息架构 / 页面（v4 仪表盘版 · 2026-07-02 四项拍板）
> 拍板记录：① 今日仪表盘置顶（3 秒给答案）；② 通用老黄历宜忌降级为折叠条并标注「千人一面」；③ 本轮聚焦仪表盘，M7 学堂/M8②③ 后置；④ 手机+电脑双端都要好用。

1. **今日仪表盘**（首屏，永远显示今天）：主题句 + 柔和顺逆刻度（调候/顺/平/留意，不用红字分数）+ 个人化宜忌 + **谶言·丞相奏对**（谶以取象、奏以进言，非判词）+ 经典引句（带出处，按日轮换）+ 迷你时间层 + 折叠版通用老黄历；未录生辰时为录入 CTA。
2. **万年历**：年月快跳（1901–2099）；录入后整月**顺逆点**（调候鎏金/顺石绿/留意赭空心）+「本月大势」摘要；选定日面板含时间层叠加与**流年/流月与命局关系速记**（大势的骨感事实）。
3. **今日详解 + 本命盘**：流日体性诗（滴天髓）、关系连线图（同型同字合并、按类型共道）、合并解读列表 + 宫位尾注一次化；四柱表/五行环/旺衰用神/大运条。
4. **录入生辰**：公历、时辰、性别；（真太阳时留二期）。
5. **学堂 / 易学**：术语点开能学 + 后天八卦图（M7 将扩为工作坊）。
6. 置顶锚点导航：今日 · 万年历 · 命盘 · 学堂 · 易学。

## 5. 技术架构（网页优先 · 大陆可直连）
- **零构建的纯静态网页**：`index.html` + `styles.css` + 几个 vanilla JS 文件，双击即可打开看效果，不需要打包工具、不需要服务器、不需要 npm 才能运行。
- **无任何海外运行时依赖**（大陆访问的硬约束）：
  - 命理库 `lunar-javascript` **本地 vendor** 到 `vendor/lunar.js`，不走 CDN（jsDelivr / unpkg 在大陆不稳定）。
  - **不用 Google Fonts / gstatic**，只用大陆设备自带的系统字体（PingFang SC / 微软雅黑 / 宋体 / 楷体）。
  - 不接 Google Analytics / reCAPTCHA 等被墙服务，不加载任何海外资源。
- 目录（扁平、易懂）：
  - `vendor/lunar.js` 本地命理库（唯一第三方代码，已落地到本机）
  - `js/engine.js` 排盘封装（包住 lunar 全局对象 → 导出干净数据）
  - `js/interpret.js` 化学反应规则 + 中文文案
  - `js/app.js` 表单、渲染、localStorage
  - `styles.css` 素静样式
- 数据：生辰存 `localStorage`，可一键清除；全程不联网、不上传。
- 部署：整个文件夹就是成品，丢到任意静态托管即可（大陆可放国内对象存储 / OSS + CDN）。
- 备注：`package.json` 仅用于开发时拉取 / 更新命理库并 vendor，**运行成品不需要它**。将来若体量变大需要工程化，再引入构建工具；现在以"先看到效果"为先。

## 6. 美学方向（待确认，下为默认建议）
- **素静文人风（默认推荐）**：米白宣纸底、墨色字、衬线中文标题、单一低饱和点缀色（黛 / 赭），大量留白。呼应"不贩卖 FOMO"的一贯声纹，以及雍正"素而贵"、Diptyque 的克制高级感。
- 备选：传统黄历红金（不推荐，易俗）；现代极简卡片（可作素静风的骨架）。
- 配色策略（v2 更新）：**容器仍素静**（宣纸底、墨字、留白），但**五行与关系改用鲜明功能色**——五行各一色、合冲刑害各一色，为「一目了然」服务；饱和度提高但仍克制，不堆金。
